# youtubeautoplay

Simple Expo Go app (Expo SDK 53) that automatically plays a YouTube video on launch.

## Running

1. Install dependencies with `npm install`.
2. Start the development server with `npm start`.
3. Open the project in Expo Go v53 or later; the video will play automatically.

If `npm start` fails with `Cannot find module 'metro/src/ModuleGraph/worker/importLocationsPlugin'`,
install the bundled Metro version used by Expo 53 with `npm install metro@^0.81`.

The video being played is [h51ZMOJMw-E](https://www.youtube.com/watch?v=h51ZMOJMw-E).
