# youtubeautoplay

Simple Expo Go app that automatically plays a YouTube video on launch.

## Running

1. Install dependencies with `npm install`.
2. Start the development server with `npm start`.
3. Open the project in Expo Go; the video will play automatically.

The video being played is [h51ZMOJMw-E](https://www.youtube.com/watch?v=h51ZMOJMw-E).
